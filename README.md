# Selma Oktaviani Portfolio

## Project Overview
A responsive, dependency-free portfolio website built to showcase UI/UX projects, quality assurance experience, and system optimization skills. The portfolio includes responsive navigation, project filtering, and accessible case-study modals.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Local Development
Since this is a plain HTML/CSS/JS project, no build tools are required.
1. Clone the repository.
2. Open `index.html` directly in your browser.
3. Alternatively, you can use a local static server like `npx serve` or Live Server in VS Code.

## Deployment
This project is ready to be deployed to GitHub Pages or Vercel. 
- **Vercel**: Import the repository in Vercel and it will automatically deploy as a static site without any build step required.
- **GitHub Pages**: Go to repository settings > Pages > select branch `main` and save.
